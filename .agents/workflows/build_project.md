Run these commands to build the Next.js project from scratch and ensure it is healthy.

// turbo
1. Install dependencies:
   ```bash
   npm install
   ```

// turbo
2. Run typecheck:
   ```bash
   npm run typecheck
   ```

// turbo
3. Build the Next.js app:
   ```bash
   npm run build
   ```
