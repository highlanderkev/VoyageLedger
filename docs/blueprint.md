# **App Name**: Voyage Ledger

## Core Features:

- Trip Planner: Enable users to create detailed trip plans with destination, dates, and budget.
- Expense Logging: Allow users to log expenses with category, amount, and date for each trip.
- Expense Summaries: Provide interactive summaries and visualizations of expenses per trip.
- Budget Tracking: Track spending against the set budget for each trip, displaying remaining funds.
- AI Expense Suggestion: An AI tool assists the user in suggesting commonly forgotten travel expenses based on destination.
- AI Trip Report Generation: Allow users to generate basic reports of their trip in plain text, to easily share their experiences on travel blog or forum.

## Style Guidelines:

- Primary color: Coastal blue (#64B5F6) to evoke travel and calmness.
- Background color: Desaturated light blue (#E3F2FD) for a calming backdrop.
- Accent color: Sandy beige (#F5DBC2) for highlights and interactive elements.
- Body and headline font: 'Inter' sans-serif for a clean and modern look.
- Use line-style icons related to travel, finance, and destinations.
- Use subtle transitions and animations for a smooth user experience.
